# Aurora

Aurora is a productivity and focus web application designed to support students with time management and concentration difficulties. The system combines a flexible Pomodoro-style timer with session tracking and productivity insights to encourage structured and intentional study habits.

The project was developed as part of a Final Year Project.

## Core Features

- Focus and break timer system
- Adjustable focus and break durations
- Circular animated timer interface
- Session tagging and module organisation
- Productivity insights dashboard
- Session history tracking
- Local storage persistence
- Responsive user interface
- Browser notification support (experimental)

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/jsultana/Aurora-react.git
```

Navigate into the project folder:

```bash
cd Aurora-react
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The project should then run locally through the Vite development server.

---

## How to Use Aurora

1. Select a focus duration and break duration.
2. Press Start to begin a focus session.
3. Pause or reset the timer if needed.
4. Add optional modules or session tags through the Settings page.
5. View completed sessions and productivity insights through the Insights dashboard.

---

## Known Limitations

- Session data is stored locally in the browser rather than through a database.
- Browser notifications may behave inconsistently depending on browser permissions and deployment environment.
- The system currently does not include user authentication or cloud synchronisation.
- Mobile responsiveness has been partially implemented but could be improved further in future iterations.
